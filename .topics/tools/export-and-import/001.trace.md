# Export and Import Root

This topic tracks interoperability work that converts source lineage or
provenance formats into Tiinex trace artifacts and, where useful, back out
again.

## Current Read

The current work here is not one format. It is a branch for format-specific
export and import slices.

That means:

- the root stays focused on interchange policy and branch structure
- source-format leaves carry concrete mapping evidence or design notes
- OpenLineage is one such leaf, not the whole program
- future source formats can be added without collapsing them into one bucket
